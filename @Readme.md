STM32F103

Librairies pour carte Bluepill à base de STM32F103

Ces librairies fonctionnent sous EmIde 2.20 ou plus généralement sous ARM-Gcc, 
Elles ont été testées avec un émulateur ST-Link-v2. 

Les librairies s'appuient généralement sur les librairies standard fournies par 
ST-Microelectronics pour les STM32F1.
Un fichier startup.s spécifique (ci-joint) est utilisé pour la gestion des interruptions, 
à placer dans le dossier Setup du projet EmIde.
